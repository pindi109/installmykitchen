# Install My Kitchen

Static website for installmykitchen.co.uk — independent kitchen installation in Coventry & Warwickshire.

## Tech Stack
- Astro 5 (static)
- Tailwind CSS
- Netlify (deploy)

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy
Deploys to Netlify on push to main branch.
